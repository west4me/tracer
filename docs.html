Overview

tracer is a specialized tool designed for QA professionals and developers to capture, document, and analyze user interactions within web applications. It provides detailed logging of user actions, accessibility information, and technical details in real-time.

Precision Tracking

Captures every interaction with exact timestamps, element details, and context.

Smart Documentation

Automatically generates comprehensive logs in multiple formats for easy sharing.

Features

Real-time Action Logging

  • Captures clicks, form inputs, navigation, and keyboard interactions
  • Records element properties, states, and accessibility attributes
  • Tracks form submissions and validation states

Error Tracking

  • Automatic console error capture with stack traces
  • Network error monitoring and reporting

Interaction Tracking

tracer captures detailed information about various types of user interactions:

Click Events

{
    "type": "click",
    "element": "button.submit-form",
    "timestamp": "2025-02-11T14:23:42",
    "context": "checkout-form",
    "details": {
        "text": "Submit Order",
        "aria-label": "Complete purchase",
        "disabled": false
    }
}

Form Interactions

{
    "type": "input-change",
    "element": "input[type='email']",
    "timestamp": "2025-02-11T14:23:45",
    "details": {
        "value": "user@example.com",

        "required": true
